# 🚆 Train Consist Management App

## 📌 Overview
The **Train Consist Management App** is a Java-based console application designed to demonstrate the practical use of **Core Java, Data Structures, Algorithms, and System Design concepts** through a real-world railway scenario.

The project simulates how train bogies are managed, validated, processed, and optimized using modern programming techniques.

---

## 🎯 Objective
This project aims to bridge the gap between **theoretical data structures** and **real-world application design** by:

- Applying Java Collections Framework
- Demonstrating algorithm efficiency
- Handling real-world constraints (validation, safety)
- Showcasing clean and scalable design practices
